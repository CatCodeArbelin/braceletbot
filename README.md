# braceletbot

Телеграм-бот для оформления заказов на браслеты.

## Локальный запуск (Windows + Docker Desktop)

1. Скопируйте файл переменных окружения:
   ```bash
   cp .env.example .env
   ```
2. Откройте `.env` и заполните переменную `BOT_TOKEN`.
3. Запустите проект:
   ```bash
   docker compose up --build
   ```

## Деплой на Ubuntu VDS

1. Клонируйте репозиторий:
   ```bash
   git clone <repo>
   ```
2. Перейдите в папку проекта:
   ```bash
   cd <repo>
   ```
3. Скопируйте шаблон переменных окружения:
   ```bash
   cp .env.example .env
   ```
4. Отредактируйте переменные окружения:
   ```bash
   nano .env
   ```
5. Запустите контейнер в фоне:
   ```bash
   docker compose up -d --build
   ```

## Эксплуатация

- Просмотр логов:
  ```bash
  docker compose logs -f
  ```
- Остановка контейнеров:
  ```bash
  docker compose down
  ```

## Сценарий заказа и хранение данных

Пользователь в боте выбирает товар из каталога, указывает параметры/детали заказа, оставляет контактные данные и подтверждает оформление. После подтверждения заказ сохраняется в SQLite-базе по пути:

`/app/data/orders.db` (в контейнере), что соответствует файлу `./data/orders.db` на хосте благодаря маунту `./data:/app/data`.
